# 🎤 AI Voice Telegram Bot

An AI powered voice chatbot built using **n8n automation** and **ElevenLabs text-to-speech**.

This bot can listen to voice messages, generate AI responses, and reply back with voice.

---

## 🚀 Features

✨ Voice to Text conversion  
🤖 AI generated responses  
🔊 Text to Speech using ElevenLabs  
📩 Automatic reply on Telegram  
⚡ Fully automated workflow using n8n  

---

## 🛠️ Tech Stack

- n8n (Workflow Automation)
- ElevenLabs (Text to Speech AI)
- Telegram Bot API
- AI Agent

## ⚙️ Setup Instructions

1. Import **workflow.json** in n8n  
2. Add your **ElevenLabs API key**  
3. Connect your **Telegram Bot**  
4. Run the workflow  

---

## 🧠 How It Works

```
Telegram Voice Message
        ↓
Speech to Text
        ↓
AI Agent Response
        ↓
ElevenLabs Text to Speech
        ↓
Voice Reply to Telegram
```
